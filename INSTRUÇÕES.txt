====================================
  COMO USAR
====================================

1️⃣ INSTALAR
   cd HOME
   npm install

2️⃣ DESENVOLVER
   npm run dev
   → http://localhost:3000

3️⃣ GERAR SITE ESTÁTICO
   npm run build
   → Pasta: out/

====================================
  DEPLOY (ESCOLHA 1)
====================================

🚀 VERCEL (30 segundos)
   npm i -g vercel
   vercel --prod

🚀 NETLIFY (drag & drop)
   https://app.netlify.com/drop
   → Arraste a pasta "out/"

🚀 GITHUB PAGES
   git push
   → Settings > Pages > Source: main/out

🚀 QUALQUER SERVIDOR
   → Upload da pasta "out/"
